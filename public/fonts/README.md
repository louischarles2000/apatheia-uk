# Fonts Directory

This directory contains local font files used in the Apatheia UK website.

## Required Fonts

You need to add the following font files to this directory:

- georgia.woff2 - Georgia Regular font
- georgia-bold.woff2 - Georgia Bold font

You can convert existing TTF or OTF fonts to WOFF2 format using online converters like Font Squirrel's Webfont Generator: https://www.fontsquirrel.com/tools/webfont-generator

## Font Licensing

Please ensure you have the proper licensing for any fonts used in this project.
